## Project Description
Care-Bot is a NAO V6 humanoid robot programme designed to assist 
elderly residents in a care home in Wales. The programme coordinates 
daily schedules using speech recognition, head touch sensing and LED 
visual feedback.

## Robot
NAO V6 - Choregraphe 2.8.7.4

## How to Run
1. Open Choregraphe 2.8.7.4
2. Connect to NAO V6 robot
3. Create a new Python Script box
4. Paste the code from carebot.py
5. Connect Robot onStart to Python box onStart
6. Connect Python box onStopped to Robot onStopped
7. Press play

- Head touch sensor to start the programme
- Time based activity announcement
- Full daily schedule reading
- Speech recognition for yes, no, repeat, stop commands
